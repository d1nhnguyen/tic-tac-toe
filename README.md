# React Tic-Tac-Toe App

A simple **React application** that lets users play Tic-Tac-Toe.

### Features
- Dynamic 3x3 board built with React loops  
- Move history with coordinates (row, col)  
- Shows current move: *"You are at move #..."*  
- Sort moves (ascending / descending)  
- Highlights winning line and shows draw message  

###  Tech & Structure
Built using **React functional components** and **Hooks (`useState`)**.  
Main components: `Square.jsx`, `Board.jsx`, `Game.jsx`  
Each file is clean, formatted, and commented.
